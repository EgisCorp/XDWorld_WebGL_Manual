# JSVec2Array

new Module.JSVec2Array\(\) 로 생성합니다.

```javascript
var vec_array = new Module.JSVec2Array();
```

## new Module.JSVec2Array\(\) → JSVec2Array

> 새로운 JSVector2D 배열 객체를 생성합니다.

{% tabs %}
{% tab title="Information" %}
* Return : JSVector2D 배열 객체 [JSVec2Array](jsvector2d.md)
{% endtab %}
{% endtabs %}
