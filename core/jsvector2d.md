# JSVector2D

> new Module.JSVector2D API 생성.
> 
> JSVector2D 생성 시 기본값 x, y(0, 0) 설정.

```javascript
var position = new Module.JSVector2D(129.128265, 35.171834);

var position = new Module.JSVector2D();
```

## properties

| Name | Type | Description |
| :--- | :--- | :--- |
| x | number | x 좌표 값 |
| y | number | y 좌표 값 |
